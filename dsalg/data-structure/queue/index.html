# 队列

Queue 队列

# 定义

  • 队列是一个有序的线性表,可以用数组或者链表实现
  • 队列遵循先进先出(LILO:first in first out) 原则,即现存的数据先出队列,后存的数据后出队列

# 示例

  • ①图中:空队列的时候队头和队尾都指向默认的一个节点

  • ③图中:出队同入队只需要挪动队头的位置即可,及改变front变量的值
  • ③④图中:当队尾达到存储空间尾部即代表队列已满
